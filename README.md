A powerful tool to detect duplicate and similar code in JavaScript functions.

## Installation

`ash
npm install -g find-duplicate-js
`

## Usage

`ash
find-duplicates ./src
find-duplicates-ui
`

## Features

- Smart function detection
- Interactive web UI
- Configurable similarity threshold
- Zero dependencies
